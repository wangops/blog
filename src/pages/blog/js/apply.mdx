#### Javascript apply、call、bind
###### &nbsp;
apply、call 和 bind 是用于控制函数执行时 this 上下文的方法。它们都是 Function 原型的一部分，虽然它们的目的类似，但用法略有不同。
###### &nbsp;
#### Javascript apply、call、bind
###### &nbsp;
#### 1 call、apply 和 bind 使用规则
###### &nbsp;
#### （1） call
###### &nbsp;
&nbsp;&nbsp;&nbsp;&nbsp;  目的：以特定的 this 值调用函数，并逐个传递参数
###### &nbsp;
&nbsp;&nbsp;&nbsp;&nbsp;  语法：func.call(thisArg, arg1, arg2, ...)
###### &nbsp;
<SyntaxHighlighter language="javascript">
  {`
        function greet(greeting, name) {
        console.log(greeting, name);
        }

        greet.call(null, 'Hello', 'Alice');  // 输出: "Hello, Alice!"
`}
</SyntaxHighlighter>
###### &nbsp;

#### （2） apply
###### &nbsp;
&nbsp;&nbsp;&nbsp;&nbsp;  目的：以特定的 this 值调用函数，并将参数作为数组（或类数组对象）传递。
###### &nbsp;
&nbsp;&nbsp;&nbsp;&nbsp;  语法：func.apply(thisArg, [arg1, arg2, ...])
###### &nbsp;
<SyntaxHighlighter language="javascript">
  {`
    function greet(greeting, name) {
        console.log(greeting, name);
    }

    greet.apply(null, ['Hello', 'Bob']);  // 输出: "Hello, Bob!"

`}
</SyntaxHighlighter>
###### &nbsp;

#### （2） bind
###### &nbsp;
&nbsp;&nbsp;&nbsp;&nbsp;  目的：创建一个新的函数，该函数在调用时将 this 关键字绑定到指定的值，并将给定的参数序列放在任何传递给新函数的参数之前。
###### &nbsp;
&nbsp;&nbsp;&nbsp;&nbsp;  语法：const newFunc = func.bind(thisArg, arg1, arg2, ...)
###### &nbsp;
<SyntaxHighlighter language="javascript">
  {`
    function greet(greeting, name) {
        console.log(greeting, name);
    }

    const greetHello = greet.bind(null, 'Hello');
    greetHello('Charlie');  // 输出: "Hello, Charlie!"

`}
</SyntaxHighlighter>
###### &nbsp;

#### 2 主要区别
###### &nbsp;
&nbsp;&nbsp;&nbsp;&nbsp; call 和 apply 立即执行函数。
###### &nbsp;
&nbsp;&nbsp;&nbsp;&nbsp; bind 返回一个新函数，不立即执行。
###### &nbsp;
&nbsp;&nbsp;&nbsp;&nbsp; call 逐个接收参数。
###### &nbsp;
&nbsp;&nbsp;&nbsp;&nbsp; apply 将参数作为数组接收。
###### &nbsp;
&nbsp;&nbsp;&nbsp;&nbsp; bind 允许预设一些参数，并延迟执行。
